# SIO Science Continuity and Strategy

A single-page Scripps working document on the current threats to federal science and the moves available to the SIO community.

**Live:** https://sdiggs.github.io/sio-strategy/

Working draft. Monochrome while it iterates; oxblood and teal styling to come. Every factual claim links to a primary source. A few links are still being verified, and two coordination channels (the contribution drive and the SIO Signal group) are placeholders until their URLs are set.

No build step. One self-contained HTML file with inline CSS and two D3 force graphics loaded from a CDN. Edit, commit, and GitHub Pages serves it.

S.C.D.
